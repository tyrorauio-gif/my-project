## 📝 Как добавить новую новость

1. Скопируйте `news/template.html` → `news/article4.html`
2. Заполните дату, заголовок и текст статьи
3. В `news.html` добавьте блок с кратким описанием:

```html
<div class="news-item">
    <span class="news-date">📅 ДД ММММ ГГГГ</span>
    <h3>Заголовок новости</h3>
    <p>Краткое описание...</p>
    <div class="news-footer">
        <a href="news/article4.html" class="read-more">Читать далее →</a>
    </div>
</div>
